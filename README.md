# df-explorer

This is a **VERY WIP** tool for interactively building Dockerfiles.
The goal is to let you define the Dockerfile steps from within the container itself.
This allows for exploration and testing as you write each call.

Currently the tool only contains the most basic features, allowing you to invoke things like RUN or ENV from within the container and have them reflected in the base dockerfile.
